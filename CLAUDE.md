# penman

Markdown → platform-styled rich text on the clipboard. CLI + MCP server + Claude Code plugin. See README.md for usage.

## Design Context
- **Register**: brand (marketing/brag-doc site)
- **Purpose**: convert Claude Code users into installs — the page itself is proof penman's typographic obsession is real
- **Audience**: devs in the Claude Code plugin ecosystem
- **Personality**: dry, precise, craftsman
- **DNA**: Red Pencil — Neo-Brutalist + Editorial Minimalism type voice (Creator archetype, Sage inflection) — see DESIGN.md
- **Constraints**: static single-page HTML/CSS, no framework, published to upubli.sh; WCAG AA (tokens contrast-checked)
- DESIGN.md is law: re-read it before any site implementation or review pass; deviations edit DESIGN.md first, then code
